# Todo App

A clean, lightweight, and intuitive Todo Application built to keep tasks organized and productivity high. 

## Preview

## Features
- Add new tasks effortlessly
- Edit existing tasks
- Remove tasks with a click
- Mark tasks as completed/incompleted
- Filter tasks by status (All / Active / Completed)
- Clean, responsive UI optimised for both desktop & mobile
- Fast and smooth interactions
